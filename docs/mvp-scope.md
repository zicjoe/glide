# Glide MVP Scope

## MVP must-have

- Merchant profile creation
- Default settlement asset selection
- Payout destination management
- Treasury bucket configuration
- Invoice creation
- Checkout page
- Quote preview
- Payment confirmation flow
- Settlement in sBTC or USDCx
- Post-settlement allocation
- Merchant dashboard with settlement history

## MVP plus

- Idle balance hold or earn toggle
- One simplified auto-deployment path
- Settlement detail page
- Basic refund workflow
- Basic reconciliation record

## Out of scope for first build

- Full production route execution
- Advanced multi-strategy yield management
- Off-ramp integrations
- Team roles and permissions
- Detailed analytics exports

## Demo path

1. Merchant connects wallet
2. Merchant configures settlement asset
3. Merchant adds payout destinations
4. Merchant sets treasury split policy
5. Merchant creates invoice
6. Customer opens checkout
7. Customer pays
8. Glide settles payment
9. Glide allocates funds across buckets
10. Dashboard updates with settlement outcome
